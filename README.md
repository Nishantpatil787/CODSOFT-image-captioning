# Image Captioning AI

This project generates captions for images using a transformer-based model (BLIP).

## Features
- Automatic caption generation
- Works on multiple images
- Uses deep learning (CV + NLP)

## Tech Stack
- Python
- Transformers (BLIP)
- PyTorch

## How to Run
1. Install requirements
2. Run main.py

Image → Caption
